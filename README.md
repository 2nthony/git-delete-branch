## Features

- Support delete matched branches. e.g.`fix/*`

## Prerequires

- git
- node.js

## Install

```console
yarn global add git-delete-branch
```

## Usage

Basically:

```console
git-delete-branch [...branches]
```

Or using git external commands:

```console
git delete-branch [...branches]
```

## CLI

`git-delete-branch [...branches] [options]`

### `branches`

Delete branches

### `options`

#### `-r, --remote` (WIP)

Delete remote branches
